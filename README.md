# Better Screenshot

A Raycast extension for organizing and managing screenshots.

## Getting Started

1. Bind the **Take Screenshot** command to your preferred shortcut
2. Take a screenshot and select a base directory for saving
3. Confirm the save location

That's it. Each time you take a screenshot you'll be prompted to choose where within your base directory it gets saved.

## Planned Features

- Full directory navigation
- Change your preferred base directory
- Rename screenshots
- Create new directories from within the extension
- File format conversion
- and more
